# UniversalScriptRblx
Roblox Testing Utility

This script was created for Roblox game testing and development purposes only. It is intended to help developers test gameplay mechanics, movement systems, and map interactions during development. It is not designed or intended for exploiting games, bypassing security systems, or gaining unfair advantages in public experiences.

## Main Features :
Infinite Jump
WalkSpeed Adjustment
JumpPower Adjustment
No Clip
